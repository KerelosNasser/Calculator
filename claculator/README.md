## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/KerelosNasser/Calculator.git
   cd Calculator
   ```

2. Install dependencies
   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

## Keyboard Shortcuts

- Numbers: `0-9`
- Operations: `+`, `-`, `*`, `/`
- Equals: `=` or `Enter`
- Clear: `Escape`
- Backspace: `Backspace`
- Decimal point: `.`
